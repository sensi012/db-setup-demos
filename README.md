# Azure Database Setup Demos

This repository contains Infrastructure as Code (IaC) demonstrations and templates for provisioning various database services on Microsoft Azure.

## Available Setups

- **[Cosmos DB](./cosmos-db-setup)**: Deployment configuration for Azure Cosmos DB.
- **[MySQL](./mysql-db-setup)**: Deployment configuration for Azure Database for MySQL.
- **[Redis](./redis-db-setup)**: Deployment configuration for Azure Cache for Redis.

## Usage

Navigate into any of the setup directories to find the specific deployment templates and parameter files required to provision the respective database resources. Each directory serves as a standalone reference for its corresponding database technology.
